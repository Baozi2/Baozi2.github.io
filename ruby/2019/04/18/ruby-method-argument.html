可使用的参数：

  1. 必须参数 args
  2. 可选参数 args=4
  3. 变长参数 **args
  4. 关键字参数 args: args: 4
  5. block参数(将传入的block装换为一个Proc对象)

    这些参数推荐的顺序为 必选参数-> 可选参数-> 变长参数 -> 关键词参数 -> 变长关键词参数 -> block参数

注意：

  1. block参数必需为最后一个参数
  2. 关键词参数放于普通参数之后
  3. 变长关键词参数放于关键词参数之后
  4. 变长参数放于可选参数之后
  5. 变长参数可以放于必选参数前中后

以下为一个例子:

def method(a, b=2, *c, d:, e: 5, **f, &block)
  p a
  p b
  p c
  p d
  p e
  p f
  p block
end
