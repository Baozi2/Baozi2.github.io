Introduction

centos6.7内核升级

1 查看当前内核版本

$ uname -r
$ cat /etc/redhat-release ##查看centos版本

2 导入key

$ rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org

3 安装ELRepo

$ rpm -Uvh http://www.elrepo.org/elrepo-release-6-6.el6.elrepo.noarch.rpm

4 安装长期支持版本的内核

$ yum --enablerepo=elrepo-kernel install kernel-lt -y

5 修改默认启动的内核

vi /etc/grub.conf

新安装的内核一般在第一个，把defaut=1改为0即可

default=1
timeout=5
splashimage=(hd0,0)/boot/grub/splash.xpm.gz
hiddenmenu
title CentOS (3.10.94-1.el6.elrepo.x86_64)
root (hd0,0)
kernel /boot/vmlinuz-3.10.94-1.el6.elrepo.x86_64 ro root=UUID=3f8f9044-c1d5-4412-8470-87d810bbdb15 rd_NO_LUKS rd_NO_LVM LANG=en_US.UTF-8 rd_NO_MD SYSFONT=latarcyrheb-sun16 crashkernel=auto KEYBOARDTYPE=pc KEYTABLE=us rd_NO_DM rhgb quiet console=tty0 console=ttyS0,115200n8
initrd /boot/initramfs-3.10.94-1.el6.elrepo.x86_64.img
title CentOS release 6.7 (Final) 2.6.32-431.el6.x86_64 mod
root (hd0,0)
kernel /boot/vmlinuz-2.6.32-431.el6.x86_64-mod ro root=UUID=3f8f9044-c1d5-4412-8470-87d810bbdb15 rd_NO_LUKS rd_NO_LVM LANG=en_US.UTF-8 rd_NO_MD SYSFONT=latarcyrheb-sun16 crashkernel=auto KEYBOARDTYPE=pc KEYTABLE=us rd_NO_DM rhgb quiet console=tty0 console=ttyS0,115200n8

6 重启

$ reboot 或者 shutdown -r now

$ uname -r #查看当前内核版本
